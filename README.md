# YouTube Creator Website

A modern website for YouTube creators built with Next.js and TypeScript.

## Features

- **Header with Logo and Search Bar**: Easily navigate and search through content
- **Editable About Me Section**: Customizable information about the creator
- **Videos Display**: Showcase your latest video uploads
- **Shorts Display**: Display your YouTube Shorts content
- **Subscriptions**: Show channels you're subscribed to
- **Worlds & Modpacks**: Download section for Minecraft worlds and modpacks
- **Admin Mode**: Easily add new content through an admin panel

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Build for production:
```bash
npm run build
npm start
```

## Admin Features

Click the Admin button in the header to enable Admin Mode. This allows you to:
- Add new videos
- Add new shorts
- Add new downloads (worlds & modpacks)
- Add new subscriptions
- Edit the About Me section
